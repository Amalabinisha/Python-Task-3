# News Headlines Web Scraper

## Overview
Scrapes headlines from BBC News using Python (requests, BeautifulSoup).  
Saves all headlines (from `<h2>` and `<h3>` tags) into `headlines.txt`.

## Requirements
- Python 3.x
- requests (`pip install requests`)
- beautifulsoup4 (`pip install beautifulsoup4`)

## Usage
1. Run `news_scraper.py`.
2. Headlines will be saved in `headlines.txt`.
3. Headlines are also printed in the console.

## Files
- news_scraper.py — Python script
- headlines.txt — Output file (generated)
